Building an 8 bit TTL computer, while frugal

<yt> <ia>

For a long time, I have wanted to build a TTL CPU from scratch, but I simply can't get around to it. Maybe because it's because of my horrible procrastination habits, or the fact that I'm living with limited money. However, I've now decided to design one in hopes that one day, I'm able to build one if I have the money to build it.

The Specs

The specs are:

  • 8-bit data width
  • 16-bit address width
  • 1 Accumulator, 1 Index register, and 1 register that fuctions as both
  • 74382 ALU
  • Control logic uses PLDs
  • 32K+(?) ROM
  • 128K RAM

The block diagram:

  • A : Accumulator
  • B : GP register + Index register
  • C : Index register
  • PM : Pointer Multiplexer(chooses which register is the index)
  • + : Adder (for indexing)
  • AR : Address register
  • SP : Stack Pointer
  • PC : Program Counter
  • SU : Shifter Unit
  • AM : ALU Multiplexer(chooses which register to operate)
  • ALU : Arithmetic Logic Unit
  • AL : ALU Latch
  • IR : Instruction Register
  • CU : Control Unit
  • AB : Address Buffer
  • DB : Data Buffer
